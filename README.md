# Student-GPA-Tracker
Developed a responsive Student GPA Tracker using HTML, CSS, and JavaScript. Features include SGPA and CGPA calculation, dynamic subject management, grade-point conversion, interactive analytics.
A simple and user-friendly web application that helps students calculate their **SGPA (Semester Grade Point Average)** and **CGPA (Cumulative Grade Point Average)** quickly and accurately.

## 🚀 Features

- Calculate SGPA based on subject credits and grades
- Calculate CGPA across multiple semesters
- Instant and accurate results
- Simple and responsive user interface
- Easy-to-use form inputs
- Works on desktop and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📖 How It Works

### SGPA Calculation

SGPA is calculated using the formula:

```
SGPA = Σ(Credit × Grade Point) / Σ(Credits)
```

### CGPA Calculation

CGPA is calculated using the formula:

```
CGPA = Σ(All Semester Grade Points) / Total Semesters
```
